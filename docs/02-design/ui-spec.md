### 반응형 브레이크포인트

- Tablet: 768px ~ 1023px

### 2.3 주제 선택 페이지 `/select-topic`

```
+--------------------------------------------------+
|  [←] 뒤로          2/4 단계          [로고]     |
+--------------------------------------------------+
|  [진행 바: ●●○○]                                |
|                                                  |
|  [π 수학] 탐구 주제를 선택하세요                |
|                                                  |
|  추천 주제                                       |
|  ┌────────────────────────────────────────────┐  |
|  │ ○ 피타고라스 정리의 실생활 응용            │  |
|  └────────────────────────────────────────────┘  |
|  ┌────────────────────────────────────────────┐  |
|  │ ○ 확률과 통계로 분석하는 스포츠 데이터     │  |
|  └────────────────────────────────────────────┘  |
|  ┌────────────────────────────────────────────┐  |
|  │ ○ 미적분의 개념과 물리학의 연관성          │  |
|  └────────────────────────────────────────────┘  |
|  ┌────────────────────────────────────────────┐  |
|  │ ○ 황금비율과 자연 및 예술 속 수학적 원리   │  |
|  └────────────────────────────────────────────┘  |
|  ┌────────────────────────────────────────────┐  |
|  │ ○ 암호학의 수학적 기초와 RSA 알고리즘      │  |
|  └────────────────────────────────────────────┘  |
|                                                  |
|  ── 또는 직접 입력 ──────────────────────────   |
|  ┌────────────────────────────────────────────┐  |
|  │ 탐구하고 싶은 주제를 직접 입력하세요...    │  |
|  └────────────────────────────────────────────┘  |
|                                                  |
|         ┌──────────────────────────┐             |
|         │   다음 →  (비활성)       │             |
|         └──────────────────────────┘             |
+--------------------------------------------------+
```

**동작**
- 추천 주제 클릭 → 선택 (라디오 방식, 1개만 선택 가능)
- 직접 입력 필드에 텍스트 입력 시 추천 주제 선택 해제
- 주제 선택 또는 직접 입력(1자 이상) 시 "다음" 버튼 활성화

**선택 상태**
```
미선택: │ ○ 피타고라스 정리...  │  bg: white
선택됨: │ ● 피타고라스 정리...  │  bg: #EEF2FF, border: #4F46E5
```

## 3. 공통 UI 컴포넌트

### 3.1 Header / Navigation Bar
```
+--------------------------------------------------+
| [←] 뒤로    제목 또는 로고    단계 표시(N/4)     |
+--------------------------------------------------+
```
- 랜딩 페이지에서는 뒤로가기 숨김
- 생성 중 페이지에서는 뒤로가기 비활성

### 3.2 Progress Bar
```
[●●●○]  단계 표시 (점 + 선)
████░░░  퍼센트 바 (생성 중 페이지)
```

### 3.3 Toast Notification
```
+─────────────────────────────+
│ ✓ 클립보드에 복사되었습니다 │  (Success: green, 하단 중앙, 3초)
+─────────────────────────────+

+─────────────────────────────+
│ ✗ 생성 중 오류가 발생했습니다│  (Error: red, 재시도 링크 포함)
+─────────────────────────────+
```

### 3.4 모바일 레이아웃 (320px)
- 카드 그리드: 2열 (과목 선택) → 1열 (주제 선택)
- 옵션 버튼: 가로 3개 → 세로 스크롤
- 결과 텍스트 박스: 전체 너비, 스크롤 가능

## 4. 접근성 기준

- 색상 대비: Primary `#4F46E5` on white = 5.1:1 (WCAG AA 통과)
- 키보드 탐색: Tab 순서 명확, 선택 상태 focus ring 표시
- ARIA: `role="radiogroup"`, `aria-selected`, `aria-live="polite"` (생성 중/완료)
- 이미지 alt 속성 필수
