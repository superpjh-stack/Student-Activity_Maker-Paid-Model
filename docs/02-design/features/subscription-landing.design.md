# AI 생기부 Maker — 랜딩 & 구독 전환 UX 기획서

**버전**: 1.0 | **작성일**: 2026-03-19

---

## 1. 디자인 방향성

### 컬러 시스템 (기존 연속성 유지)
```
Primary:   #c084fc (fuchsia-400)
Secondary: #e879a0 (pink-500)
Accent:    #f9a8d4 (pink-300)
BG:        #fdf6fa
Glass:     rgba(255,255,255,0.88)
Dark:      #3d1535
```

### 디자인 원칙
- **Z세대 감성**: 글래스모피즘 + 그라디언트 + 플로팅 애니메이션
- **불안/FOMO 유발**: "지금 247명이 생성 중" 실시간 느낌
- **신뢰 구축**: 구체적 수치, 실제 학생 언어의 후기
- **최소 마찰**: CTA 클릭 → 모달 → 결제 3단계

---

## 2. 페이지 섹션 구성

### 섹션 1: Hero

**헤드카피**: "대입의 20%가 / 지금 이 순간 결정된다"
- 기존 "AI가 써준다" 방식보다 위기감/중요성 강조
- 20%는 학종 비율을 암시하는 앵커링

**서브카피**: "AI가 3분 만에 완성하는 세특 500자 · 탐구보고서 / 지금 경쟁자도 쓰고 있습니다"

**CTA 버튼**: "지금 무료로 시작하기 →" (btn-gradient, 56px height, rounded-2xl)

**소셜 프루프 Pills** (Hero 하단):
- 🎓 고등학생 12,400명+
- 📝 세특 38,000건+ 생성
- ⭐ 만족도 4.9/5.0

**긴급성 배지** (우측 상단 pulse 애니메이션):
- "지금도 247명이 세특 생성 중" (실시간 느낌 주는 ping 애니메이션)

**우측 프리뷰 카드**: 실제 생성 중인 세특 텍스트 타이핑 애니메이션 (animate-float)

### 섹션 2: Pain Point (공감 섹션)

**3개 Glass Card**:

| 아이콘 | 제목 | 설명 |
|--------|------|------|
| 😩 | 뭘 써야 할지 모르겠어 | 책은 읽었는데 500자가 안 나와… |
| ⏰ | 제출 기한은 내일인데 | 밤 11시에 아직 한 글자도 못 썼어… |
| 😤 | 써도 어색하고 틀린 것 같아 | 친구 거 보면 훨씬 잘 쓴 것 같아… |

**섹션 하단 전환**: "이 3가지 고통, AI 생기부 Maker가 한 번에 해결합니다 →"

---

### 섹션 3: Before/After 비교 (데모)

**Before (좌측, 흐릿 처리)**:
- 어색한 학생 글체로 미완성 세특 보여줌
- "2시간 후에도 미완성" 레이블

**After (우측, 강조)**:
- 완성된 생명과학 세특 (실험 데이터 포함)
- 태그: "500자 완성" / "진로연계" / "실험데이터"
- "⏱ 2분 47초" 표시

**CTA**: "나도 저렇게 쓰고 싶어 →"

---

### 섹션 4: 기능 Bento Grid

6개 카드 (2×3 그리드, 일부 wide):
1. ✍️ 세특 500자 자동생성 (Wide)
2. 📑 탐구보고서
3. 🎯 진로연계 단락
4. 📦 배치 생성 (Wide)

### 섹션 5: 후기 + 통계

**후기 카드 3개** (실제 학생 말투):
- 고2 수학 학생: "진짜 나 이거 없었으면 세특 포기했을 것 같음"
- 고3 생명과학: "탐구보고서 실험 데이터까지 넣어줘서 진짜 놀람"
- 고1 국어: "처음엔 반신반의했는데 진짜 유용함"

**통계 카운터 배너**:
- 12,400명+ 누적 이용
- 38,000건+ 생성
- 2분 53초 평균 완성

---

### 섹션 6: 프라이싱 (핵심 전환 섹션)

**헤드**: "입시 한 달에 커피 두 잔값"

**월간/연간 토글** (연간: 2개월 무료, 가격 ~33% 할인):

| 플랜 | 월간 | 연간(월환산) | 특징 |
|------|------|------------|------|
| Free | 0원 | — | 일 3회 제한 |
| **Pro** ⭐ | 9,900원 | 6,583원 | 무제한 생성 (추천 배지) |
| Family | 19,900원 | 12,416원 | 자녀 3명 + 학부모 확인 |

**Pro 카드 디자인**: 중앙 배치, scale(1.03) 확대, 그라디언트 배경, "가장 인기" 뱃지

**하단 신뢰 요소**: "💳 7일 이내 전액 환불 보장 · 언제든 해지 가능 · 카드/카카오페이"

## 3. 구독 전환 플로우 UX

### 업셀 모달 (Paywall Hit)
트리거: 무료 3회 소진 시 자동 노출

**모달 내용**:
```
"오늘 무료 생성 3건을 모두 사용했어요"
Pro 플랜 혜택 리스트
[Pro 구독하기 — 월 9,900원] → 결제
[내일 다시 올게요] → 닫기
7일 무조건 환불 보장
```

### 결제 페이지 최소 마찰 (4필드)
1. 이름
2. 이메일
3. 결제수단 (카드 / 카카오페이 선택)
4. 약관 동의 체크박스 1개

### 구독 완료 후 온보딩 (3스텝)
1. "🎉 Pro 구독 완료! 첫 세특을 생성해볼까요?"
2. 과목 선택 가이드
3. 완성 → "선생님께 바로 전달하기" 버튼 노출

---

## 4. 애니메이션 & 인터랙션

| 요소 | 애니메이션 | 효과 |
|------|-----------|------|
| Hero 카드 | animate-float (3s ease-in-out) | 생동감 |
| 세특 텍스트 | typewriter (2.5s steps) | 실시간 생성 느낌 |
| 섹션 진입 | scroll reveal (translateY 24px → 0) | 자연스러운 등장 |
| 버튼 hover | translateY(-2px) + shadow 증가 | 클릭 유도 |
| 통계 숫자 | countUp fade-in | 신뢰감 |

---

## 5. 반응형 고려사항

- **모바일**: Hero 세로 레이아웃, 프라이싱 카드 단열, Bento wide → 1열
- **태블릿**: 2열 그리드 유지
- **PC**: max-w-5xl 중앙 정렬, 3열 그리드
