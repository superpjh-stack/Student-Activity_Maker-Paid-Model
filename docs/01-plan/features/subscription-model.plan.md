## 2. 핵심 수익화 전략

### 프리미엄 유료화 핵심 기능 3선

#### 기능 1: 무제한 생성 + 배치 생성
- **선정 이유**: 고3 학생의 실제 니즈는 "8과목 전부 한번에" → 배치 생성이 없으면 서비스 가치의 70% 미활용
- **업셀 타이밍**: 3회 소진 직후, 배치 생성 버튼 클릭 시

#### 기능 2: 합격생 세특 스타일 분석 (SKY 익명)
- **선정 이유**: 경쟁사 복제 불가한 고유 데이터 자산 → "내 세특이 SKY 합격생 수준인가?" 강력한 FOMO
- **데이터 수집**: 자발적 익명 기여 프로그램 (합격 인증 후 제공 동의)
- **업셀 타이밍**: 세특 생성 완료 후 "합격생은 어떻게 썼을까?" CTA 노출

#### 기능 3: 담임 선생님 공유 링크
- **선정 이유**: B2B2C 확장의 교두보 — 선생님이 학생에게 서비스 추천하는 역전 현상 유도
- **기능 상세**: 공유 링크 → 선생님이 코멘트 추가 → 학생에게 푸시 알림
- **업셀 타이밍**: "선생님께 바로 전달하고 싶다면 입시생 플랜으로"

---

### 무료 → 유료 전환 트리거 설계

```
무료 3회 소진
  → 소진 직전(2회) 배너: "이번 달 1회 남았어요. 지금 업그레이드하면 20회!"
  → 소진 시 모달: "다음 생성은 {다음달 1일}에 가능합니다. or 지금 바로 계속하기 →"
  → 7일 후 미전환 시 이메일: "세특 마감이 언제예요? 지금 준비하세요"

프리미엄 기능 클릭
  → 잠금 아이콘 + "입시생 플랜 전용 기능입니다"
  → 30초 미리보기 후 블러 처리
  → "첫 달 50% 할인으로 시작하기" CTA
```

## 3. 창의적 차별화 요소

### 경쟁사 대비 독창적 구독 혜택

1. **세특 완성도 AI 점수** (100점 만점)
   - 생성 후 "창의성 85점, 진로 연계 72점, 구체성 90점" 자동 평가
   - 입시생 플랜은 "개선 제안 3가지"까지 제공

2. **나만의 세특 스타일 리포트** (월 1회 자동 발송)
   - "지난달 당신의 세특 패턴: 과학 탐구 중심, 진로 연계 강함"
   - 취약 과목/영역 자동 감지 후 추천

3. **진로 로드맵 연동**
   - 희망 대학/학과 입력 → 해당 학과 합격생이 강조한 세특 키워드 자동 반영
   - "서울대 생명과학부 합격생은 이 키워드를 5번 이상 사용했어요"

### B2B2C 학교/선생님 협업 모델

- 담임 선생님 전용 무료 계정 제공 (학생 관리 목적)
- 선생님이 학생 세특에 피드백 → 학생 알림 → 재생성
- 학교 내 "AI 세특 도우미" 공식 도구 포지셔닝

### 입시 컨설턴트 파트너십

- 컨설턴트 전용 "에이전시 플랜" 개발
- 컨설턴트가 학생에게 서비스 추천 시 수수료 지급 (20%)
- 컨설턴트 브랜딩 커스터마이징 (화이트라벨 검토)

## 4. 성장 지표 & OKR (6개월 목표)

### Objective: "AI 세특의 표준 도구로 자리잡는다"

| Key Result | 목표 | 측정 방법 |
|------------|------|---------|
| **MAU** | 10,000명 | Google Analytics |
| **유료 전환율** | 8% (무료 → 유료) | 내부 DB |
| **ARR** | 6,000만 원 (보수적) | Stripe/결제 데이터 |
| **B2B 계약** | 5건 이상 | 영업 CRM |
| **NPS** | 50 이상 | 월 1회 인앱 서베이 |

### 핵심 메트릭 3가지

1. **Paywall Hit Rate** (무료 소진 → 구독 페이지 이동률)
   - 목표: 30% | 현재 예상 기준점: 0%
   - 개선 수단: 업셀 모달 A/B 테스트

2. **3개월 구독 유지율**
   - 목표: 75% | 교육 SaaS 평균: 65%
   - 개선 수단: 교육 캘린더 알림, 세특 완성도 리포트

3. **과목당 평균 생성 횟수**
   - 목표: 3회 이상 (1회 생성 후 이탈 방지)
   - 개선 수단: A/B 비교, 코칭 기능, 재생성 유도 UX
