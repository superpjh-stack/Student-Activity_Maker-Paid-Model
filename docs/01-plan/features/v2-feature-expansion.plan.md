> 작성일: 2026-03-05

## 1. 배경 및 목표

### 현황 (v1)
- 8개 과목 x 5개 주제 → 탐구보고서 + 세특 500자 자동 생성
- 길이(단/중/장) x 톤(학술/친근/중립) 옵션
- 로컬 히스토리 저장, docx 다운로드
- Cloud Run 배포 완료

### v2 비전
> "단순 생성기(Generator)에서 학생의 성장 파트너(Growth Partner)로"

학생이 스스로 탐구 동기를 발견하고, 더 깊이 있는 학습으로 이어지며, 입시까지 체계적으로 연결되는 종합 플랫폼으로 진화한다.

## 2. CTO팀 기획 세션 결과

### 기획 Agent A — 사용자 경험 강화 아이디어

#### A-1. 나만의 세특 프로필 (학생 맞춤화)
- 학생이 1회 입력: 관심 진로, 희망 학과, 특기, 독서 이력
- 이후 생성 시 항상 해당 프로필에 맞게 세특/보고서를 개인화
- 예: "AI/컴퓨터공학 지망 → 모든 탐구에 IT 연계 표현 자동 삽입"

#### A-2. 단계별 탐구 코칭 (인터랙티브 플로우)
- 단순 버튼 선택 → 대화형 질문 방식으로 전환
- "이 주제를 선택한 이유가 있나요?" → 입력 → 세특 동기 파트에 자동 반영
- "실제로 해본 활동이 있나요?" → 입력 → 실제활동 파트 풍부화

#### A-3. 세특 교사 스타일 뱅크
- 실제 세특 문체 패턴 10가지 저장 (공식적/격려형/분석형 등)
- 학생이 선호 스타일 선택 → AI가 해당 문체로 생성

#### A-4. 비교 보기 & A/B 생성
- 동일 주제로 2가지 버전 동시 생성
- 나란히 비교 후 원하는 섹션만 골라 조합 (믹스앤매치)

---

### 기획 Agent B — AI 기술 혁신 아이디어

#### B-1. 마인드맵 탐구 주제 확장기
- 학생이 키워드 1개 입력 (예: "기후변화")
- AI가 8개 과목 관점에서 연결되는 탐구 주제 20개 마인드맵으로 시각화
- 과목 연계 흥미를 자극하는 "탐구 아이디어 전시관" 역할

#### B-2. AI 피드백 루프 (내 글 vs AI 글 비교)
- 학생이 직접 쓴 세특/보고서 텍스트를 붙여넣기
- AI가 항목별 점수 + 개선 제안 + 수정 버전 제공
- "교사 관점 리뷰" 모드: 어색한 표현, 논리 비약 지적

#### B-3. 진로 로드맵 연계 추천
- 생성된 탐구주제 → "이 탐구를 한 학생이 합격한 대학/학과" 데이터 연동
- 수능 탐구 영역과의 연계 힌트 제공
- "다음에 이 주제도 탐구해보세요" 시퀀스 추천

#### B-4. 실시간 스트리밍 생성 + 인라인 편집
- 생성 중 텍스트가 실시간으로 나타나는 스트리밍 UX
- 생성 완료 후 인라인 에디터로 바로 수정 가능
- 수정 내용 저장 → 재생성 시 수정 기록 반영

#### B-5. 멀티 과목 세트 생성 (배치 모드)
- 여러 과목 동시 선택 → 한 번에 여러 과목 세트 생성
- "학기말 세특 세트" 패키지 개념

---

### 기획 Agent C — 서비스 성장 & 수익화 아이디어

#### C-1. 공유 & 커뮤니티 플랫폼
- 익명 탐구 갤러리: 다른 학생들의 우수 보고서 구조 참고
- "이 주제 인기 순위" 실시간 트렌드 차트
- 학교별/지역별 탐구 주제 트렌드 분석

#### C-2. 교사용 대시보드 (B2B 확장)
- 학급 전체 세특 일괄 검토 기능
- 학생별 탐구 활동 이력 관리
- 반복 표현/표절 유사도 검사

#### C-3. 포트폴리오 생성기
- 학생의 1년 탐구 활동 → 자기소개서 연계 포트폴리오 자동 생성
- PDF 출력: "나의 탐구 여정" 형태의 입시 포트폴리오 문서

#### C-4. 알림 & 세특 시즌 캘린더
- 학기별 세특 마감일 알림 (학교 커리큘럼 기반)
- "세특 시즌이 2주 후입니다! 미리 준비하세요" 푸시 알림
- 탐구 주제 일일 아이디어 이메일 구독

#### C-5. 프리미엄 기능 (Freemium 모델)
- Free: 월 5회 생성, 단/중 길이만
- Premium: 무제한 생성, 장 길이, A/B 비교, 포트폴리오, 배치 모드

## 4. v2 Sprint 1 범위 (P0 — 즉시 구현)

### Sprint 1 목표
> "생성 경험을 극적으로 개선하여 사용자 만족도 최대화"

#### 기능 1: 스트리밍 생성 + 인라인 편집 (B-4)
- **현재**: 10초 기다린 후 결과 표시
- **변경**: 실시간 타이핑 효과로 텍스트 스트리밍
- **추가**: 생성 완료 후 텍스트 직접 수정 가능한 contenteditable 에디터

#### 기능 2: A/B 비교 생성 (A-4)
- **현재**: 1개 결과만 생성
- **변경**: "다른 버전도 보기" 버튼 → 2번째 버전 생성
- **추가**: 섹션별 토글로 A/B 선택 조합

#### 기능 3: 나만의 프로필 맞춤화 기본 (A-1 Lite)
- 진로/관심사 1줄 입력 필드 (선택사항)
- 입력 시 세특/보고서에 진로 연계 표현 강화

## 5. v2 Sprint 2 범위 (P1 — 다음 스프린트)

### 기능 4: 대화형 코칭 플로우 (A-2)
- 주제 선택 후 "탐구 배경 인터뷰" 3문답 (선택사항)
- 입력한 답변을 세특 동기/활동 파트에 자연스럽게 통합

### 기능 5: 마인드맵 주제 확장기 (B-1)
- 키워드 입력 → 과목 연계 주제 제안 시각화
- 탐구 아이디어 갤러리로 진입점 다양화

### 기능 6: AI 내 글 피드백 (B-2)
- 학생 작성 글 붙여넣기 → AI 점수 + 개선 제안
- "교사 관점 리뷰" 모드

## 6. 기술 아키텍처 방향 (아키텍트 팀에 전달)

### 스트리밍
- OpenAI Streaming API 활용 (`stream: true`)
- Next.js App Router의 `ReadableStream` + `StreamingTextResponse`

### 프로필 맞춤화
- localStorage에 프로필 JSON 저장 (비로그인 우선)
- 추후 계정 연동 시 서버 마이그레이션

### A/B 생성
- 2개 API 호출 병렬 처리 (`Promise.all`)
- 결과 비교 UI 컴포넌트 신규 개발

### 대화형 코칭
- 멀티스텝 폼 (React state machine 또는 Zustand)
- 답변 데이터를 프롬프트에 동적 주입
