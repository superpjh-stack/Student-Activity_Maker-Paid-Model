## 1. Overview

### 1.1 Purpose

단순 AI 작문 도구를 넘어, 대한민국 고등학생이 입시 준비 전 과정에서 **매일 다시 찾아오는 불가결한 파트너**로 포지셔닝한다.

현재 서비스는 탐구보고서와 세특 500자를 생성해 준다. 그러나 학생이 이 서비스를 이탈하지 않고 구독을 유지하게 만들 "심리적 잠금(Emotional Lock-in)"이 부족하다. 킬러 기능은 다음 두 가지를 동시에 달성해야 한다.

1. **고통 제거**: 입시 불안, 자기 표현 어려움, 품질 불확실성
2. **보상 제공**: 즉각적 성취감, 차별화 증거, 사회적 증명

### 1.2 Background

- 대한민국 고등학생 180만 명 중 약 40만 명이 생기부 작성에 실질적 어려움을 겪는다.
- 현재 경쟁 서비스(ChatGPT 직접 사용 포함)와의 가장 큰 차이는 "교과 맥락의 깊이"와 "입시 문서 특화성"이다.
- 구독 모델(9,900원/19,900원)은 이미 존재하나, 유료 전환율을 높이기 위한 "유료여야만 가능한 경험"이 명확하지 않다.
- 기존 기획(authentic-experience F-15~F-19, v2-feature-expansion)은 생성 품질 향상에 집중했다. 이번 기획은 **유지율과 전환율을 끌어올리는 감성 경험**에 집중한다.

### KF-01: 내 생기부 건강 진단 리포트

**태그라인**: "지금 내 생기부, 입시에서 살아남을 수 있나요?"

#### 왜 킬러인가? (학생 심리 근거)

> 고3 학생의 가장 큰 공포는 "내가 모르는 사이에 뒤처지고 있는 것"이다.
> 성적 석차는 눈에 보이지만 생기부 품질은 측정이 불가능하다.
> 이 공포에 구체적인 수치를 붙여주는 순간 학생은 즉각 행동한다.

**기능 상세**:

학생이 현재 작성된(또는 AI로 생성된) 세특/탐구보고서를 붙여넣으면, AI가 다음 7가지 항목을 점수화하여 진단 리포트를 생성한다.

| 진단 항목 | 측정 기준 | 가중치 |
|-----------|-----------|--------|
| 진정성 점수 | 1인칭 경험 서술, 구체적 에피소드 비율 | 25% |
| 전공 연계성 | 희망 학과 키워드 밀도, 진로 연결 명확성 | 20% |
| 논리 구조 | 도입-전개-결론 완결성, 인과관계 명확성 | 15% |
| 심화 탐구 깊이 | 단순 요약 vs 자기 해석/확장 비율 | 20% |
| 언어 다양성 | 반복 단어 빈도, 문장 구조 다양성 | 10% |
| AI 감지 리스크 | AI 특유 표현 패턴 비율 (역설적으로 탈AI화 필요) | 10% |
| 분량/완성도 | 권장 분량 대비 충족률 | 가중치 없음 (Pass/Fail) |

**UI/UX 흐름**:
```
[진단 시작] 버튼
    ↓
텍스트 붙여넣기 영역 (또는 이전 생성 기록에서 불러오기)
    ↓
3초 로딩 ("AI가 입시관 관점으로 검토 중...")
    ↓
종합 점수: 78점 / 100점
    ↓
항목별 레이더 차트 (6각형)
    ↓
취약 항목 상위 2개 → "AI로 즉시 보완하기" 버튼
```

**감성 카피**:
- 점수 60점 이하: "솔직히 말하면, 지금 이 세특은 입시에서 기억에 남지 않을 확률이 높아요."
- 점수 80점 이상: "이 정도면 사정관의 시선을 멈출 수 있어요. 한 단계만 더!"

**수익화 연계**:
- Free: 종합 점수만 표시 (항목별 세부 점수 잠금)
- 준비생(9,900원): 항목별 점수 + 상위 2개 취약점 설명
- 입시생(19,900원): 전체 항목 + "즉시 보완 생성" + 월 3회 진단

### KF-02: AI 입시관 (Admissions Officer Mode)

**태그라인**: "대학 입학사정관의 눈으로 네 세특을 읽어줄게"

#### 왜 킬러인가? (학생 심리 근거)

> 선생님에게 세특 검토를 부탁하는 것은 눈치 보이고, 부모님은 기준을 모른다.
> "객관적인 제3자의 전문 평가"에 대한 수요는 매우 높지만 공급이 없다.
> 입시 컨설팅 1회 비용 5~10만 원을 구독 가격에 포함시켜 버리는 것이다.

**기능 상세**:

AI가 "연세대학교 의예과 입학사정관"처럼 학과 + 학교 컨텍스트를 가진 페르소나로 세특을 평가한다.

```
평가 프롬프트 구조:
- 학생이 선택: 희망 대학 + 희망 학과
- AI 페르소나: "{학과} 전공 입학사정관 10년 경력"
- 평가 관점:
  ① 이 학과에서 원하는 역량이 드러나는가?
  ② 다른 지원자와 차별화되는 고유성이 있는가?
  ③ 학업 의지와 지적 호기심이 느껴지는가?
  ④ 사정관이 "면접에서 더 물어보고 싶다"고 느낄 포인트가 있는가?
```

**출력 예시**:
```
[연세대 의예과 AI 사정관 평가]

"이 학생의 세특에서 가장 인상적인 부분은 실험 과정의 시행착오를
솔직하게 서술한 점입니다. 그러나 '의학'과의 연결이 결론부에서만
등장하고, 탐구 전 과정에서 의사로서의 관점이 보이지 않아 아쉽습니다.

면접에서 물어보고 싶은 것: '실험 실패 경험이 의사결정에 어떤 영향을
주었나요?' — 이 질문에 답할 수 있도록 세특을 다듬어 보세요."

[즉시 수정 제안] → [AI로 보완하기]
```

**감성 포인트**:
- "면접에서 물어보고 싶다"는 표현이 학생에게 강력한 동기 부여
- 부모님께 보여드릴 수 있는 "공식적인 AI 평가서" 역할
- PDF 다운로드 → 선생님 상담 전 사전 준비 자료로 활용

**수익화 연계**:
- Free: 기능 미제공 (맛보기 1회 체험만)
- 준비생(9,900원): 월 3회 AI 입시관 평가
- 입시생(19,900원): 무제한 + 복수 대학 동시 비교 평가

### KF-03: 세특 포트폴리오 타임라인

**태그라인**: "고1부터 고3까지, 내 성장의 증거를 한눈에"

#### 왜 킬러인가? (학생 심리 근거)

> 고3 수시 원서 작성 시 가장 많이 하는 말: "내가 고1 때 뭘 했더라?"
> 학생들은 자신의 과거 활동을 기억하지 못한다. 기억 못하면 활용도 못한다.
> "내 성장 스토리를 AI가 정리해 줬다"는 경험은 서비스 락인의 가장 강력한 형태다.

**기능 상세**:

서비스에서 생성한 모든 세특/탐구보고서를 타임라인 형태로 시각화하고, AI가 "성장 내러티브"를 자동 생성한다.

**타임라인 UI**:
```
[고1 1학기] ────────────────────────────────────────── [고3 2학기]
    |               |               |               |
  수학             물리             화학            생명과학
  세특 ✅          탐구 ✅          탐구 ✅         세특 ✅
  "함수의 극한"    "파동 공명"     "산화반응"       "세포분열"
```

**AI 성장 내러티브 기능**:
```
[내 성장 스토리 생성] 버튼 클릭 →

"고1 수학에서 극한 개념을 탐구하며 수학적 사고의 엄밀함에 눈을 떴던
당신은, 고2에서 물리 파동 실험을 통해 이 엄밀함이 자연현상에도 적용
된다는 것을 발견했습니다. 고3 생명과학 세포분열 탐구에서는 수학-물리-
생물의 연결고리를 스스로 찾아냈습니다. 이 흐름은 의공학과 지망생에게
매우 설득력 있는 성장 스토리입니다."
```

**활용 시나리오**:
- 자기소개서 소재 발굴 ("내 탐구 히스토리에서 가장 강한 스토리는?")
- 면접 준비 ("내 고1~고3 탐구 흐름을 설명해 주세요")
- 선생님 상담 자료 ("선생님, 제 세특 방향성이 이렇게 이어지고 있어요")

**수익화 연계**:
- Free: 최근 3개 기록만 열람 (타임라인 잠금)
- 준비생(9,900원): 전체 타임라인 + 과목별 정리
- 입시생(19,900원): 성장 내러티브 AI 생성 + 자기소개서 연계 버튼

### KF-04: 주제 희소성 스코어 + 나만의 각도 찾기

**태그라인**: "10만 명이 쓴 '광합성 탐구', 당신은 어떻게 달라질 건가요?"

#### 왜 킬러인가? (학생 심리 근거)

> 입시에서 '이 주제를 탐구한 학생'이 1,000명이면 사정관은 지루하다.
> '이 각도로 탐구한 학생'이 처음이면 사정관은 메모한다.
> 학생들은 주제의 희소성 개념 자체를 모른다. 알게 되면 불안이 폭발한다.

**기능 상세**:

학생이 주제를 선택하면, AI가 해당 주제의 "예상 경쟁 밀도"와 "차별화 각도 5가지"를 제시한다.

**희소성 스코어 UI**:
```
선택 주제: [생명과학 > 광합성과 에너지 전환]

주제 경쟁 밀도: ████████░░ 높음 (추정 동일 주제 탐구자 多)

차별화 각도 추천:
1. 🔬 "수직 농업 환경에서 인공광 파장별 광합성 효율 비교"
   → 농업공학, 식품공학 지망생에게 적합 | 경쟁 밀도: 낮음

2. 💊 "광감각제를 이용한 암 치료(PDT)에서의 광합성 원리 응용"
   → 의예과, 약학 지망생에게 적합 | 경쟁 밀도: 매우 낮음

3. 🤖 "식물 광수용체 메커니즘의 바이오센서 응용 가능성"
   → 전자공학, 생명공학 지망생에게 적합 | 경쟁 밀도: 낮음

4. 🌿 "다양한 스트레스 조건(고온, 건조)에서 C4 식물의 적응 전략"
   → 환경학, 생태학 지망생에게 적합 | 경쟁 밀도: 중간

5. ⚡ "태양전지 효율 개선을 위한 광합성 전자 전달계 모방 연구"
   → 에너지공학, 화학공학 지망생에게 적합 | 경쟁 밀도: 낮음

[이 각도로 탐구보고서 생성하기]
```

**주제 은행과의 시너지**:
- 기존 과목별 주제 은행(고1/고2/고3 × 5과목 × 150개)을 "희소성 메타데이터"와 연결
- 인기 주제에는 🔴 "경쟁 높음" 배지, 희소 주제에는 🟢 "차별화 유리" 배지

**수익화 연계**:
- Free: 희소성 경고만 표시 ("이 주제는 경쟁이 높을 수 있어요")
- 준비생(9,900원): 차별화 각도 2개 제안
- 입시생(19,900원): 차별화 각도 5개 + 진로 연계 맞춤 정렬

### KF-05: 나만의 진로 키워드 DNA

**태그라인**: "당신을 설명하는 단어들, AI가 발굴해 드립니다"

#### 왜 킬러인가? (학생 심리 근거)

> "나는 어떤 사람인가?"를 글로 써야 하는데 모르겠다.
> 이것이 자기소개서, 세특, 면접의 공통 어려움이다.
> AI가 내 탐구 히스토리를 분석해서 "당신의 학문적 정체성"을 언어로 만들어 주면,
> 학생은 처음으로 자기 자신을 언어로 인식하는 경험을 한다. 이건 강렬하다.

**기능 상세**:

학생이 서비스에서 생성한 세특/탐구보고서 전체를 AI가 분석하여 "진로 키워드 DNA 카드"를 생성한다.

**DNA 카드 UI**:
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  홍길동님의 학문적 정체성
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  핵심 키워드 DNA
  [데이터 분석]  [에너지 전환]  [사회 문제 해결]
  [수리적 직관]  [실험 설계]

  당신은 이런 사람입니다:
  "수치 뒤에 숨겨진 패턴을 찾고, 그 패턴을 사회 문제
  해결에 연결하고 싶어하는 분석형 탐구자입니다.
  특히 에너지 분야에서 강한 지적 호기심을 보이며,
  이는 환경공학 또는 에너지경제학과와 잘 맞는
  탐구 스타일입니다."

  → 한국에너지공단 청소년 탐구 공모전 참가
  → 에너지 관련 R&E 주제 추천 3가지 보기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  [카드 저장]  [카카오톡 공유]  [자소서에 활용하기]
```

**"자소서에 활용하기" 연계 기능**:
- DNA 카드의 핵심 키워드가 자기소개서 1번 문항 초안 생성에 자동 주입
- "나는 ~한 사람입니다"로 시작하는 자기소개서 첫 문장 5가지 생성

**소셜 공유 전략**:
- 카카오톡으로 공유 → "친구도 분석받기" 링크 포함 → 바이럴 획득
- 인스타그램 스토리용 정사각형 카드 이미지 다운로드

**수익화 연계**:
- Free: 핵심 키워드 3개만 표시 (설명 텍스트 잠금)
- 준비생(9,900원): DNA 카드 전체 + 추천 활동
- 입시생(19,900원): 자소서 연계 초안 생성 + 카드 무제한 재생성

## 5. 기능 우선순위 매트릭스

임팩트(유저 가치) × 구현 난이도 × 수익 기여도 기준

| 기능 | 임팩트 | 구현 난이도 | 수익 기여 | 우선순위 점수 | 순위 |
|------|--------|-------------|-----------|---------------|------|
| KF-01 건강 진단 리포트 | 9/10 | 낮음 | 9/10 | **A+** | 1위 |
| KF-04 희소성 스코어 | 8/10 | 낮음 | 7/10 | **A** | 2위 |
| KF-05 진로 키워드 DNA | 9/10 | 중간 | 8/10 | **A** | 3위 |
| KF-02 AI 입시관 | 10/10 | 중간 | 10/10 | **A** | 4위 |
| KF-03 포트폴리오 타임라인 | 8/10 | 높음 | 7/10 | **B+** | 5위 |

**구현 권장 순서**:
```
Phase 1 (즉시, 2주):
  KF-01 건강 진단 + KF-04 희소성 스코어
  → 기존 generate 페이지 확장, 새 API 라우트 추가

Phase 2 (3~4주):
  KF-05 진로 키워드 DNA
  → 기존 history 기록 활용, DNA 카드 컴포넌트 신규 개발

Phase 3 (5~8주):
  KF-02 AI 입시관
  → 복잡한 페르소나 프롬프팅, 대학/학과 DB 구성 필요

Phase 4 (9~12주):
  KF-03 포트폴리오 타임라인
  → DB 스키마 변경, 히스토리 마이그레이션 필요
```

## 8. Risks and Mitigation

| 리스크 | 임팩트 | 발생 가능성 | 대응 방안 |
|--------|--------|-------------|-----------|
| AI 진단 점수의 신뢰성 문제 ("AI가 틀렸어요") | High | Medium | 면책 문구 + "참고용 가이드" 포지셔닝, 사용자 피드백 수집 |
| 희소성 스코어의 실제 데이터 부재 (추정치임) | Medium | High | "예상 경쟁 밀도" 표현 사용, 절대적 수치 제시 자제 |
| DNA 카드 소셜 공유 → 개인정보 노출 우려 | High | Low | 공유 전 미리보기 + 개인 식별 정보 제외 옵션 제공 |
| AI 입시관 페르소나의 대학별 정확도 | High | Medium | 특정 대학 보증 금지 문구, "AI 시뮬레이션" 명시 |
| 기능 과부하 → 핵심 생성 기능 사용성 저하 | Medium | Medium | 메인 플로우 우선, 킬러 기능은 대시보드 별도 섹션 |
