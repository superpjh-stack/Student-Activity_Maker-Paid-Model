# AI 생기부 Maker — 바이럴 성장 & 에코시스템 전략 기획서

> **Summary**: "학교 반 전체가 쓰게 만드는" 네트워크 효과 기반 바이럴 성장 전략. 학생-선생님-학원을 연결하는 B2B2C 에코시스템과 6개월 MAU 50,000명 달성 로드맵.
>
> **Project**: AI 생기부 Maker
> **Author**: PM (바이럴 성장 전략)
> **Date**: 2026-03-24
> **Status**: Draft

---

## 1. 개요 (Overview)

### 1.1 핵심 명제

대한민국 고등학생 130만 명 중 세특 작성에 실질적 도움을 받는 AI 도구는 아직 없다. 기존 경쟁 서비스(뤼튼, ChatGPT)는 범용 AI이며 세특/탐구보고서에 특화되지 않았다. AI 생기부 Maker는 **수험생 맥락에 최적화된 도구**로서, 올바른 바이럴 메커니즘을 갖추면 반 전체 전파가 가능한 구조를 만들 수 있다.

### 1.2 목표

- **6개월 후 MAU 50,000명** (현재 대비 50배 성장 가정)
- 유료 전환율 8% 이상 유지
- K-12 교육 채널(학교/학원/커뮤니티) 내 자연 바이럴 루프 3개 이상 작동

### 1.3 배경 분석

현재 구독 모델(탐구생/준비생/입시생)과 기능 세트(세특 생성, 탐구보고서, 배치 생성, 참고문헌)는 핵심 가치 제안으로 충분하다. 그러나 **성장 채널**과 **입소문 구조**가 부재하다. 이 문서는 그 격차를 메우는 전략을 다룬다.

## 2. 경쟁사 분석 및 차별화

### 2.1 경쟁 지형

| 서비스 | 주요 강점 | 세특 특화도 | 바이럴 구조 | 약점 |
|--------|----------|-----------|-----------|------|
| **ChatGPT** | 범용 AI, 인지도 최고 | 없음 (직접 프롬프트 필요) | 없음 | 세특 맥락 무지, 할루시네이션 위험 |
| **뤼튼** | 한국어 특화, 무료 | 낮음 (세특 템플릿 일부) | SNS 마케팅 | 대입 전문성 부족, 추천 근거 없음 |
| **클래스카드** | 학습 플래시카드, 학교 연계 | 없음 | 교사 채널 강함 | 세특/탐구 영역 미진출 |
| **AI 생기부 Maker** | 세특 전용 설계, 과목별 주제 은행, 참고문헌 자동 생성 | 최고 | **← 이 문서에서 구축** | 성장 채널 미구축 |

### 2.2 경쟁 우위 포인트

1. **세특 전용 AI 프롬프트**: 과학 실험 데이터 포함 규칙, 진로 연계 단락 자동 삽입 — 범용 AI 대비 즉각적 품질 우위
2. **과목×학년 주제 은행**: 고1/고2/고3 × 5과목 × 150개 주제, 일일 캐시 로직 — 매일 새로운 영감 제공
3. **참고문헌 자동 생성**: RISS/Scholar/NAVER 링크 자동 추가 — 학생이 직접 찾는 시간 절감
4. **배치 생성**: 8과목 동시 처리 — ChatGPT 단건 생성 대비 압도적 효율

### 2.3 Copy-proof 자산 (복제 불가 영역)

- 합격생 세특 익명 데이터베이스 (수집 시작 즉시 선점 가능)
- 담임 선생님-학생 연결 네트워크 (B2B2C 채널)
- 수험생 커뮤니티 내 인지도 (에브리타임 인지 효과)

### 전략 1: "공유하면 횟수 더 주기" — 소셜 공유 인센티브 루프

#### 메커니즘 (어떻게 바이럴이 되는가)

```
학생 A가 세특 생성 완료
    → "카카오톡/에브리타임에 공유하면 +2회 추가 증정" 팝업
    → 학생 A가 공유 (링크 클릭 시 자동 제목 "AI로 세특 5분 완성ㄷㄷ")
    → 친구 B가 링크 클릭 → 랜딩 페이지 + 1회 무료 체험
    → 친구 B가 가입 → 학생 A도 추가 1회 적립
    → 친구 B도 동일 루프 반복
```

핵심: **공유자와 피공유자 모두 이득**이 되는 양방향 인센티브. 링크에는 referral_code가 심겨 추적 가능.

#### 기능 요구사항

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-01 | 세특/탐구보고서 생성 완료 직후 공유 CTA 모달 노출 | Must |
| FR-02 | 카카오톡 공유 SDK 연동 (og:image 포함) | Must |
| FR-03 | Referral 코드 생성 및 추적 API | Must |
| FR-04 | 공유자 추가 횟수 적립 (친구 1명 가입 시 +2회) | Must |
| FR-05 | 피공유자 첫 방문 시 +1회 추가 체험 | Must |
| FR-06 | 에브리타임 공유용 텍스트 자동 생성 (카피라이팅 포함) | Should |
| FR-07 | 공유 횟수 대시보드 (내 추천인 현황) | Could |

#### 예상 지표

- 공유 전환율 목표: 생성 완료자의 20%
- 피공유자 가입 전환율: 40% (직접 링크 클릭이므로 고의도)
- K-factor 목표: 0.3 이상 (공유 1회당 0.3명 신규 유입)

### 전략 2: "선생님 채널" — B2B2C 교사 네트워크 확산

#### 메커니즘 (어떻게 바이럴이 되는가)

```
담임 선생님이 서비스를 인지 (블로그/유튜브/교사 커뮤니티)
    → 선생님 전용 무료 계정 생성 (학교 이메일 인증)
    → 학생 관리 클래스 생성 → 학생들에게 초대 링크 발송
    → 학생 30명 전원 한 번에 가입 (반 전체 가입)
    → 선생님이 학생 세특에 코멘트 추가 → 알림 → 재방문
    → "우리 선생님도 쓰는 서비스" 신뢰도 상승 → 구독 전환율 3배
```

핵심: **선생님의 권위**를 활용한 신뢰 전파. 1명의 선생님이 30명의 학생을 동시에 유입시키는 레버리지 효과.

#### 기능 요구사항

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-08 | 교사 전용 계정 (학교 도메인 이메일 인증) | Must |
| FR-09 | 클래스 생성 및 학생 초대 링크 발송 기능 | Must |
| FR-10 | 선생님 대시보드: 학생별 생성 현황, 과목 분포 | Must |
| FR-11 | 선생님 세특 코멘트 기능 (승인/수정 요청) | Must |
| FR-12 | 학생에게 선생님 코멘트 알림 (이메일/푸시) | Should |
| FR-13 | 선생님 전용 세특 가이드라인 설정 (학교 양식 커스텀) | Should |
| FR-14 | 교사 네트워크 유입을 위한 교사 커뮤니티(인디스쿨, 교육마당) 콘텐츠 마케팅 | Should |
| FR-15 | 교사 무료 계정 유지 조건: 학생 5명 이상 활성 | Could |

#### 예상 지표

- 교사 가입 1명당 평균 학생 유입: 18명 (반의 60% 참여 가정)
- 교사 채널 목표: 월 50명 신규 교사 → 월 900명 학생 유입
- 교사 채널 학생의 구독 전환율: 12% (교사 신뢰 효과, 일반 채널 8% 대비 +50%)

#### 교사 획득 전략

1. **인디스쿨/교육마당 콘텐츠 배포**: "AI 세특 작성 지도안" PDF 무료 제공 → 서비스 소개 포함
2. **유튜브 교육 채널 협업**: 교사 대상 유튜버(교육 유튜버 5~10만 구독자) PPL
3. **교육청 공문 활용**: 학교 공식 디지털 교육 도구로 등록 신청 (6개월 이후)

### 전략 3: "학원/컨설턴트 파트너 채널" — 입시 전문가 네트워크

#### 메커니즘 (어떻게 바이럴이 되는가)

```
입시 컨설턴트/학원이 학생에게 서비스를 추천
    → 컨설턴트 고유 추천 코드로 학생 가입
    → 학생 유료 전환 시 컨설턴트에게 20% 수수료 (월 9,900원 구독 → 1,980원)
    → 컨설턴트가 자신의 학생 10명에게 추천 → 월 19,800원 부수입
    → 컨설턴트가 자발적 홍보 → SNS/카페에 추천글 게시
    → "입시 전문가가 추천하는 서비스" 신뢰 효과
```

핵심: **경제적 인센티브** + **전문가 권위**의 결합. 컨설턴트가 능동적 세일즈 파워가 됨.

#### 기능 요구사항

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-16 | 파트너(컨설턴트/학원) 전용 계정 및 대시보드 | Must |
| FR-17 | 고유 추천 코드 + 추천 학생 현황 추적 | Must |
| FR-18 | 수수료 정산 자동화 (월 1회 계좌 이체) | Must |
| FR-19 | 파트너용 마케팅 키트 (카카오톡 공유 이미지, 소개글 템플릿) | Should |
| FR-20 | 화이트라벨 옵션: "○○학원 세특 AI" 브랜딩 커스텀 | Could |
| FR-21 | 대형 학원(100명 이상) 전용 캠퍼스 플랜 단체 계약 | Should |

#### 파트너 획득 전략

1. **네이버 카페 "수만휘/오르비" 타겟 광고**: 입시 컨설턴트 활동 공간
2. **직접 아웃리치**: SKY 입시학원 상위 50곳 이메일/전화 영업
3. **파트너 온보딩 웨비나**: 월 1회 "AI 세특 지도법" 무료 웨비나 → 파트너 전환 유도

#### 예상 지표

- 파트너 수 목표 (6개월): 100명 컨설턴트 + 20개 학원
- 파트너당 평균 학생 추천: 8명/월
- 파트너 채널 기여 MAU: 월 1,600명 (6개월 목표)

### 전략 4: "커뮤니티 임베딩" — 에브리타임·오르비·카카오 오픈채팅 자연 바이럴

#### 메커니즘 (어떻게 바이럴이 되는가)

```
[시드 콘텐츠 전략]
    → 에브리타임 고등학교 게시판에 "세특 AI 써봤는데 ㄹㅇ 개쩐다" 실제 후기 유도
    → 오르비 "세특/자소서" 게시판에 "AI 생기부 Maker 써보신 분?" 질문글 자연 발생
    → 카카오 오픈채팅 "고3 수시 정보방"에 서비스 공유 콘텐츠 자연 확산

[공유 트리거 설계]
    → 세특 생성 완료 후 "에브리타임에 후기 남기면 +3회 추가" 인센티브
    → 생성 결과 카드 이미지 자동 생성 (과목명 + "AI로 5분에 완성")
    → og:image 최적화로 링크 공유 시 미리보기 매력적으로

[커뮤니티 공식 계정 운영]
    → 에브리타임 내 공식 학교 계정 운영 (정보 제공 목적)
    → 오르비 세특 칼럼 연재 ("과목별 세특 작성 전략" 시리즈)
```

핵심: **수험생이 이미 모여있는 곳**에 자연스럽게 스며드는 전략. 광고가 아닌 정보 콘텐츠로 신뢰 구축.

| FR-23 | 에브리타임 공유 최적화 og:meta 태그 완성 | Must |
| FR-24 | 커뮤니티 후기 작성 인센티브 (스크린샷 인증 → 추가 횟수) | Should |
| FR-25 | 서비스 내 "이 주제로 생성한 학생 N명" 소셜 증거 노출 | Should |
| FR-26 | 오르비/에브리타임 공식 채널 운영 계정 세팅 | Should |

| 인스타그램 | "세특 카드뉴스" 릴스 | 주 3회 | 팔로워 5,000명 (3개월) |
| 유튜브 쇼츠 | "AI 세특 5분 완성" 과정 영상 | 주 2회 | 구독자 1,000명 (3개월) |

### 전략 5: "시즌 마케팅" — 입시 캘린더 기반 폭발적 수요 포착

#### 핵심 인사이트

수험생 서비스의 수요는 **균등하지 않다**. 특정 시즌에 집중되는 수요를 놓치지 않는 것이 성장의 관건이다.

#### 시즌별 상세 전략

**3월 신학기 (가장 중요한 시즌)**

```
시기: 3/2~3/31
타겟: 고1(새로운 세특 시작), 고2(작년 복습 + 새 학년), 고3(마지막 세특 시작)

핵심 메시지: "새 학년 첫 세특, AI로 완벽하게 시작하세요"

캠페인:
- "신학기 3일 무제한" 이벤트: 3/2~3/4 무료 사용자도 무제한 체험
- 학교별 TOP 3 인기 주제 공개 (익명 집계, 트렌드 데이터 활용)
- 입시생 플랜 첫 달 50% 할인 (9,950원)
- 선생님 초대 캠페인: "우리 반 담임 선생님 초대하면 학생 전원 +5회"

목표 MAU 기여: +15,000명 (신학기 급증)
```

**6월 모의고사 ~ 7월 여름방학**

```
시기: 6/1~8/31
```
