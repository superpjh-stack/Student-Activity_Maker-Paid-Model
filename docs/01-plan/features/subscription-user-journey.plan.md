# 구독 전환을 위한 사용자 여정 & 리텐션 전략

> **Summary**: AI 생기부 Maker의 무료 서비스를 구독 모델로 전환하기 위한 페르소나 설계, 사용자 여정 맵, 바이럴 획득, 리텐션, 이탈 방지, B2B 전략 기획서

---

## Overview

### 핵심 가치 제안
- **시간 절약**: 세특 1건 작성 시간 3시간 → 10분으로 단축
- **마감 스트레스 해소**: 담임 선생님 제출 기한 전 알림 및 초안 자동 생성
- **입시 경쟁력 확보**: 세특 품질이 수시 합격을 좌우한다는 불안 해소

---

## 1. 페르소나 설계

### 페르소나 A: 고3 수험생 "박수진" (17세, 경기도 분당)

**프로필**
- 수능 D-150일, 학종 위주 대학 지원 예정
- 수학/과학 세특 3건 미작성, 담임 제출 마감 2주 뒤
- 에브리타임, 인스타그램 일 2시간 이상 사용
- 과외·학원 병행으로 자유 시간 하루 1시간 미만

**핵심 고통점**
- "뭘 써야 할지 모르겠어. 주제도 없고 시간도 없어"
- "친구들은 다 썼다는데 나만 아직 못 썼어" (FOMO)
- "선생님이 수정 요청하면 또 시간이 날아가"

**구독 전환 트리거**
- 마감 압박 + 무료 한도 소진: "이번 달 남은 세특 2건인데 크레딧이 0이야"
- 가격 저항 낮음 (월 9,900원 = 학원 자습 1회)
- 부모님 카드 결제 가능 여부가 핵심 변수

---

### 페르소나 B: 고1 신입생 "김도현" (15세, 서울 노원구)

**프로필**
- 중학교 때 생기부 개념 전혀 없었음
- 담임 선생님이 "세특 주제 정해 와라" 했는데 뭔지 모름
- 유튜브 쇼츠·틱톡 주 사용층

**핵심 고통점**
- "세특을 어디서 시작해야 해? 주제 선택부터 막막해"
- "내가 쓴 게 잘 쓴 건지 모르겠어"

**구독 전환 트리거**
- 학기 초 세특 계획 시즌: 3월·9월 개강 후 2주
- "생기부 관리 플랜" 기능(프리미엄) 매력 포인트
- 부모님 "이거 괜찮은 거야?" → 학부모 페르소나 C와 연결

---

### 페르소나 C: 학부모 "이영미" (47세, 서울 강남구)

**프로필**
- 자녀 고2, 의대 지망
- 입시 컨설팅 월 50만 원 지출 중
- 네이버 카페 "입시정보 나눔방" 활동

**핵심 고통점**
- "입시 컨설팅비는 비싼데 세특 피드백은 잘 안 해줌"
- "아이가 혼자 쓴 세특이 합격 수준인지 모르겠어"

**구독 전환 트리거**
- 학부모 단톡방 신뢰 기반 추천 (소셜 프루프)
- 연간 구독 할인 제시
- "의대/SKY 합격 선배 세특 사례" 레퍼런스 공개

## 2. 사용자 여정 맵 (User Journey Map)

```
인지 → 첫 방문 → 가입 → 첫 생성 → 습관화 → 구독 전환 → 갱신/이탈 방지
```

### 감정 곡선
```
인지(50) → 첫방문(65) → 가입(55) → 첫생성(90) → 습관화(75) → 구독전환(70) → 갱신(80)
```

### 단계별 핵심 설계

| 단계 | 핵심 터치포인트 | 감정 | 개선 기회 |
|------|--------------|------|----------|
| 인지 | 에브리타임, 유튜브 쇼츠, 학부모 단톡방 | 호기심 + 반신반의 | SEO: "세특 쓰는 법" 키워드 |
| 첫 방문 | 랜딩 페이지 히어로 섹션 | 기대감 상승 | 회원가입 없이 1건 무료 체험 |
| 가입 | 첫 결과물 확인 후 CTA | "결과물이 좋네. 가입할게" | 카카오 소셜 로그인 1클릭 |
| 첫 생성 | 결과물 확인 순간 (Aha Moment) | 감탄 + 안도 | 5분 이내 첫 생성 완료 목표 |
| 습관화 | 대시보드 세특 현황 위젯 | "이제 여기서 하는 거지" | 월간 완성도 리포트 |
| 구독 전환 | 무료 크레딧 소진 + 마감 D-7 | "이 가격이면..." | 3일 무제한 체험 오퍼 |
| 갱신 | 2학기 개강 전 캠페인 | 방학 이탈 유혹 | 구독 일시 중지(Pause) 기능 |

## 3. 바이럴 & 획득 전략

### 3-1. 자연 공유 메커니즘

**결과물 워터마크 공유**
- 생성된 세특 하단에 "AI 생기부 Maker로 작성" 텍스트 + QR코드 삽입
- 학생이 선생님에게 제출하는 문서 자체가 바이럴 매체
- 프리미엄 구독자는 워터마크 제거 옵션 (전환 인센티브)

**세특 완성 인증 카드**
- 생성 완료 후 인스타그램 스토리 최적화 카드 자동 생성 (1080×1920)
- "#AI생기부메이커 #세특완성" 해시태그 자동 삽입
- 공유 시 크레딧 1건 즉시 지급

### 3-2. 친구 초대 (Refer-a-Friend)
- 초대한 친구 첫 생성 완료 시: 초대자 +2건, 신규 +1건
- 카카오톡 공유 버튼 + 6자리 추천 코드
- 메시지 템플릿: "야 나 이거 쓰는데 세특 진짜 10분 만에 완성됨ㅋㅋ 이거로 가입하면 무료 크레딧 받아"

### 3-3. 수능 D-day 연계
- D-100 특별 세트 한정 할인 오퍼
- "수능 D-100인데 세특은 완성했니?" SNS 공유 카드
- 에브리타임 고3 갤러리 "D-100 기념 무료 세특 1건" 이벤트

### 3-4. 커뮤니티 채널별 전략

| 채널 | 전략 |
|------|------|
| 에브리타임 | 고3 갤러리 자연 후기 유도, 학교별 키워드 광고 |
| 오픈카카오톡 | "고3 세특 공유방" 운영자 협력, 이벤트 공지 협찬 |
| 인스타그램 릴스 | "세특 Before/After", "AI 생성 속도 실시간 공개" |
| 유튜브 쇼츠 | "고1/2/3 세특 완전 정복 시리즈" 튜토리얼 |

## 4. 리텐션 & 습관 형성 전략

### 4-1. 알림 전략

| 시점 | 메시지 | 채널 |
|------|--------|------|
| 학기 초 (3월 1주) | "1학기 세특 계획 세울 시간이야. 지금 시작해봐!" | 이메일 + 앱 푸시 |
| 중간고사 2주 전 | "세특 초안 완성하면 선생님께 플러스야. 지금 시작해!" | 앱 푸시 |
| 마감 D-14 | "세특 제출 D-14. 미완성 과목: 수학, 영어. 지금 완성할까?" | SMS + 이메일 |
| 마감 D-3 | "(긴급) 세특 마감 3일 남았어. 지금 바로 완성하자!" | SMS + 이메일 + 푸시 |
| 방학 시작 1주 후 | "방학 중 다음 학기 세특 미리 준비하는 학생이 수시 유리해" | 이메일 |

### 4-2. 게이미피케이션

| 레벨 | 조건 | 칭호 | 혜택 |
|------|------|------|------|
| 1 | 첫 세특 생성 | 세특 새싹 | 크레딧 +1건 |
| 2 | 3개 과목 완성 | 세특 탐험가 | 고급 탐구 주제 추천 해금 |
| 3 | 5개 과목 완성 | 세특 마스터 | 전문가 피드백 1회 무료 |
| 4 | 전 과목 완성 | 생기부 완성자 | 프로필 뱃지 + 할인 쿠폰 |

- 7일 연속 접속: 크레딧 +2건
- 14일 연속: "골든 탐구자" 임시 뱃지

### 4-3. 주간 이메일 리포트 (매주 월요일 07:00)
```
제목: [AI 생기부 Maker] 수진아, 세특 3과목 남았어!

- 완성된 세특 현황 (과목별 체크)
- 이번 주 추천 탐구 주제 (과목별 3개)
- 세특 마감 D-day 카운트다운
- "이번 주 완성하면 어때?" CTA
```

### 4-4. 세특 완성 체크리스트
- 8개 과목 × 세특/탐구보고서 체크리스트
- 담임 제출 여부 기록 가능
- 완성 시 레벨업 애니메이션
- 프리미엄: PDF 내보내기 + 버전 관리

## 5. 이탈 방지 전략

### 5-1. 구독 취소 시 설득 플로우

```
1단계: "잠깐만요" 팝업
  → 남은 크레딧 소멸 경고 + [1개월 더 유지] / [취소 계속]

2단계: 취소 이유 선택
  [ ] 가격 부담  [ ] 지금 필요 없어요  [ ] 기능 미흡  [ ] 다른 서비스

3단계: 이유별 맞춤 오퍼
  - 가격 부담 → "50% 할인 1개월 더"
  - 방학 중 → "구독 일시 중지 사용하시겠어요?"
  - 기능 불만 → 피드백 수집

4단계: 취소 완료 이메일 → "재가입 첫 달 30% 할인: COMEBACK30"
```

### 5-2. 방학 중 이탈 방지

**구독 일시 중지 (Pause)** — 방학 기간 최대 2개월 허용
- 개강 2주 전 자동 재개 알림: "2학기 세특 시즌 시작돼. 구독 재개할까?"

**여름방학 특별 콘텐츠**
- "방학 중 탐구활동 주제 50선" 무료 PDF 제공
- "2학기 세특 미리 준비 챌린지": 방학 중 탐구보고서 1건 완성 시 크레딧 +3건
- "대학 탐방 후기를 세특으로 연결하는 법" 콘텐츠 시리즈

**수능 후 이탈 방지 (고3)**
- 수능 직후 "수시 면접 예상 질문 생성기" 무료 기능 (한정)
- "후배들에게 내 경험 공유하면 크레딧" 캠페인

### 5-3. 재활성화 캠페인

| 이탈 기간 | 메시지 | 채널 |
|-----------|--------|------|
| 14일 비접속 | "세특 마감 다가오는데 준비됐어?" + 미완성 과목 | 이메일 |
| 30일 비접속 | "돌아와! 새 AI 기능 업데이트. 크레딧 2건 선물" | 이메일 + SMS |
| 60일+ 비접속 | "다시 시작하면 첫 달 무료" | 이메일 |

## 6. B2B 채널 전략

### 6-1. 학원/입시컨설팅 파트너십

| 모델 | 내용 |
|------|------|
| 리셀러 | 학원이 학생에게 재판매, 마진 20% 지급 |
| 화이트라벨 | 학원 브랜드 커스터마이징 버전 |
| 레퍼럴 수수료 | 추천 링크 가입 시 첫 달 수수료 30% 지급 |

### 6-2. 담임선생님 레퍼럴 프로그램

| 구분 | 내용 |
|------|------|
| 교사 무료 계정 | 담임 선생님 무제한 조회 전용 계정 무료 제공 |
| 클래스 코드 | 코드로 학생 가입 시 → 선생님 크레딧 적립 |
| 피드백 도구 | 선생님이 학생 세특에 직접 코멘트 |
| 반 10명+ 구독 시 | 선생님에게 상품권/교육 자료 제공 |

### 6-3. 교육청/학교 공식 협약 (장기 전략)

- 6개월: 파일럿 학교 3~5개 확보
- 12개월: 시도 교육청 협약, 에듀테크 소프트랩 인증
- 24개월: 전국 학교 공식 도입

**핵심 포지셔닝**: "AI는 보조 도구, 학생이 주체" — 학생 수정 과정 UX 명시

## 8. 핵심 인사이트 3가지

**1. "Aha Moment" 설계가 구독의 전부**
가입 후 5분 이내에 세특 초안을 완성시켜야 한다. 첫 생성 완료 순간의 감탄이 월 9,900원 결제 의사를 만든다.

**2. 방학이 이탈이 아닌 다음 학기 잠금 장치가 되어야**
구독 일시 중지(Pause) 기능이 없으면 방학마다 이탈이 반복된다. 방학을 "다음 학기 세특 미리 준비하는 시간"으로 포지셔닝하는 것이 핵심.

**3. 에브리타임 자연 후기 1건이 광고 1,000만 원보다 강함**
고3 갤러리 자연 후기의 전환 효과는 유료 광고를 압도한다. 초기에는 UGC 바이럴 집중 → B2B는 사용자 기반 확보 후 진입.

## 9. 구현 우선순위 (MoSCoW)

### Must (현재 이터레이션 필수)
- [ ] 무료 크레딧 3건 + 구독 결제 플로우 (카카오페이/카드)
- [ ] 구독 취소 시 1단계 설득 팝업
- [ ] 세특 완성 체크리스트 (기본)
- [ ] 마감 D-14, D-3 이메일 알림
- [ ] 친구 초대 링크 생성

### Should (이번 분기)
- [ ] 공유 카드 자동 생성 (인스타그램 최적화)
- [ ] 게이미피케이션 레벨 시스템 (4단계)
- [ ] 주간 이메일 리포트
- [ ] 구독 일시 중지(Pause) 기능
- [ ] 수능 D-day 위젯

### Could (다음 이터레이션)
- [ ] 교사용 협업 피드백 도구
- [ ] 학원 관리 대시보드 (B2B)
- [ ] 세특 버전 관리
