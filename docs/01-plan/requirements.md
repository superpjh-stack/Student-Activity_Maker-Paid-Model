# 요구사항 명세서 - AI 생기부 Maker

## 1. 서비스 개요 및 목표

### 개요
AI 생기부 Maker는 대한민국 고등학생을 위한 세부능력특기사항(세특) 및 탐구보고서 자동생성 웹서비스입니다.

### 목표
- 학생이 자신의 관심 과목과 탐구 주제를 선택하면, AI가 세특 문장 및 탐구보고서 초안을 즉시 생성
- 학생의 학교생활기록부 작성 부담을 줄이고, 자기주도적 학습 기록을 체계적으로 지원
- 누구나 쉽게 사용할 수 있는 직관적인 UI/UX 제공

---

## 2. 타겟 사용자

- **주 대상**: 대한민국 고등학생 (1~3학년)
- **부 대상**: 학생의 세특 작성을 지원하는 교사 및 학부모
- **사용 맥락**: 방과 후, 주말, 학기말 등 세특 정리 시즌

---

## 3. 핵심 기능

### 3.1 과목 선택 UI
- 8개 과목 제공: 수학, 영어, 국어, 물리, 화학, 생물, 사회문화, 역사
- 카드 또는 버튼 형태로 과목 선택
- 하나의 과목 선택 후 다음 단계로 이동

### 3.2 세부활동 주제 후보 표시
- 선택한 과목에 맞는 탐구 주제 후보 5개 표시
- 사전 정의된 주제 풀에서 제공 (아래 3.7 참조)
- 후보 주제 클릭 시 선택

### 3.7 과목별 세부활동 주제 목록 (사전 정의)

#### 수학
1. 미적분을 활용한 실생활 최적화 문제 탐구 (비용 최소화, 이익 극대화)
2. 확률과 통계로 분석하는 스포츠 데이터 (타율, 승률, 예측 모델)
3. 수열과 점화식으로 이해하는 자연 속 피보나치 수열
4. 행렬 변환을 이용한 컴퓨터 그래픽스 원리 탐구
5. 게임 이론과 내시 균형으로 분석하는 경제적 의사결정

#### 영어
1. 영어 뉴스 기사 분석을 통한 미디어 리터러시 탐구
2. 영미 문학 속 사회적 메시지 비교 분석 (디스토피아 소설 중심)
3. 영어 광고 언어의 설득 전략 분석 (수사학적 기법 활용)
4. SNS 영어 표현의 언어학적 변화와 신조어 형성 탐구
5. 환경 관련 국제 협약 원문 독해 및 핵심 내용 정리

#### 국어
1. 현대시 속 시대적 배경과 저항 정신 분석 (일제강점기 문학 중심)
2. 소설 속 서술 기법이 독자 공감에 미치는 영향 탐구
3. 미디어 언어의 의미 변화와 프레이밍 효과 분석
4. 고전 문학과 현대 창작물의 상호텍스트성 비교 연구
5. 토론 텍스트 분석을 통한 논증 구조와 오류 유형 탐구

#### 물리
1. 롤러코스터 설계로 탐구하는 역학적 에너지 보존 법칙
2. 스마트폰 센서 데이터로 분석하는 가속도와 관성 법칙
3. 레이저와 빛의 굴절·반사 원리를 이용한 광통신 탐구
4. 핵융합 에너지의 원리와 미래 에너지원으로서의 가능성 분석
5. 드론의 양력 원리와 유체역학적 비행 메커니즘 탐구

#### 화학
1. 일상 속 산화-환원 반응 탐구 (부식, 연소, 표백 현상)
2. 플라스틱 분해 과정과 환경 오염의 화학적 메커니즘 분석
3. 천연 지시약으로 탐구하는 산-염기 반응과 pH 변화
4. 의약품 속 유기화합물의 구조와 약리 작용 탐구
5. 식품 첨가물의 화학적 성분 분석과 안전성 평가

#### 생물
1. 유전자 편집 기술(CRISPR-Cas9)의 원리와 윤리적 쟁점 탐구
2. 장내 미생물군(마이크로바이옴)이 건강에 미치는 영향 분석
3. 식물의 광합성 효율과 환경 조건(빛, CO₂, 온도) 간의 관계 탐구
4. 바이러스의 변이 메커니즘과 백신 개발 원리 탐구
5. 생태계 먹이사슬에서 키스톤 종(keystone species)의 역할 분석

#### 사회문화
1. SNS가 청소년 자아정체성 형성에 미치는 영향 탐구
2. 저출생·고령화 사회 현상의 원인과 정책적 대응 방안 분석
3. 소비 문화와 과시 소비(베블런 효과)로 본 현대 사회 탐구
4. 다문화 사회에서의 문화 충돌과 문화적 상대주의 적용 사례 분석
5. 미디어가 재현하는 젠더 이미지와 사회적 고정관념 탐구

#### 역사
1. 조선 후기 실학 사상의 형성 배경과 근대화에 미친 영향 탐구
2. 산업혁명이 유럽 사회 구조(노동, 도시화, 계층)에 미친 변화 분석
3. 냉전 시대 이데올로기 대립이 한반도 분단에 미친 영향 탐구
4. 동아시아 역사 인식 갈등(위안부, 독도, 역사 교과서)의 현황과 과제
5. 세계화 시대 민족주의의 부활 현상과 역사적 유사 사례 비교

### 3.3 직접 주제 입력 옵션
- 제공된 후보 외에 학생이 직접 탐구 주제를 입력할 수 있는 텍스트 입력 필드
- 후보 선택과 직접 입력 중 하나를 선택

### 3.4 탐구보고서 생성
- **길이 옵션 (3단계)**:
  - 단: 요약 형태, 500자 내외
  - 중: 일반 형태, 1000자 내외
  - 장: 상세 형태, 2000자 내외
- **톤 옵션 (3가지)**:
  - 학술적: 논문체, 전문 용어 활용
  - 친근한: 구어체, 읽기 쉬운 표현
  - 중립적: 일반적인 보고서 문체

### 3.5 세특 500자 생성
- 고정 분량: 500자 (± 10자 허용)
- **필수 구조 (4단계 흐름)**:
  1. 동기: 해당 주제를 탐구하게 된 계기
  2. 실제활동: 구체적으로 수행한 탐구 활동 내용
  3. 결과해석: 탐구 결과 및 본인의 해석/인사이트
  4. 타학문/진로연계: 다른 학문 분야 또는 진로와의 연결
- 교사가 직접 작성한 것처럼 자연스러운 문체

### 3.6 결과 복사 기능
- 생성된 탐구보고서 및 세특 내용을 클립보드에 복사
- 복사 완료 시 시각적 피드백 (버튼 상태 변경 또는 토스트 메시지)

## 4. 사용자 플로우

```
[시작 페이지]
    ↓ "시작하기" 버튼 클릭
[과목 선택 페이지]
    ↓ 8개 과목 중 1개 선택
[탐구 주제 선택 페이지]
    ↓ 후보 5개 중 선택 OR 직접 입력
[생성 옵션 설정 페이지]
    ↓ 길이(단/중/장) + 톤(학술적/친근한/중립적) 선택
[결과 생성 중 페이지]
    ↓ AI 생성 로딩 (스피너/프로그레스바)
[결과 확인 페이지]
    ↓ 탐구보고서 + 세특 500자 표시
    ↓ 각 섹션별 복사 버튼
[다시 시작 or 다른 과목 선택]
```

## 5. 페이지 목록 및 역할

| 페이지 | 경로 | 역할 |
|--------|------|------|
| 시작(랜딩) 페이지 | `/` | 서비스 소개, 시작하기 버튼 |
| 과목 선택 페이지 | `/select-subject` | 8개 과목 카드 UI |
| 주제 선택 페이지 | `/select-topic` | 주제 후보 5개 + 직접 입력 |
| 옵션 설정 페이지 | `/set-options` | 길이 및 톤 선택 |
| 생성 중 페이지 | `/generating` | 로딩 인디케이터 |
| 결과 페이지 | `/result` | 탐구보고서 + 세특 결과 표시 및 복사 |

## 7. 비기능 요구사항

### 성능
- 세특 및 탐구보고서 생성 응답 시간: 10초 이내 (목표 5초 이내)
- 페이지 초기 로딩: 3초 이내

### 사용성
- 모바일 퍼스트 반응형 디자인 (320px ~ 1440px)
- 접근성: 키보드 탐색 가능, 명도 대비 WCAG AA 기준 준수
- 한국어 전용 UI

### 보안
- 학생 개인정보 수집 최소화 (회원가입 불필요)
- AI API 키는 서버 사이드에서 관리, 클라이언트에 노출 금지

### 안정성
- AI 생성 실패 시 사용자에게 명확한 오류 메시지 표시
- 재시도 기능 제공

### 기술 스택 (예상)
- Frontend: Next.js (React) + Tailwind CSS
- AI: Claude API (Anthropic)
- 배포: Vercel 또는 정적 호스팅
